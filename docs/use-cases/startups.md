## Common Startup Scenarios

### Scenario 1: First 10 Hires

**Challenge**: Rapidly onboarding early employees while maintaining security

**Solution**:
1. Set up your workspace in [5 minutes](../getting-started/initial-setup.md)

3. Assign basic roles (Admin, Member, Guest)

- Team management

## Startup Success Tips

### 1. Start Simple
Don't over-engineer your security. Start with basics and add complexity as you grow.

### 2. Document Everything
Create an internal security handbook. Your future team will thank you.

### 3. Automate Onboarding
Set up templates for new hire access. Save hours on each new employee.

### 4. Review Quarterly
As you pivot and grow, review your access structure every quarter.

### 5. Think Ahead
Choose settings that will scale. Changing mid-growth is painful.
